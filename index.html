Hover over a station to see the number of bikes

About this map

This is a live view of Nextbike stations in Olomouc, Czech Republic. The circle size and colour show relative numbers of bikes between all stations.

  • Data source: Nextbike API documentation
  • Refresh: approximately once per minute
  • Interaction: hover over a station to see its current bike count

This is an experimental GIS project for exploring bike-share patterns in Olomouc.
